DISCLAIMER

This is a subset of "The Caltech-UCSD Birds-200-2011 Dataset" and therfore the same terms and conditions apply.
See http://www.vision.caltech.edu/visipedia for more information.

This dataset contains 2999 training images and 2903 for testing spread over 100 classes.

-----
DATA DESCRIPTION
----------

images.txt Maps image ids to the path of the file:  <image id>, <image path>

image_class_labels.txt Contains the class id labeling of each image:  <image id>, <class id>

classes.txt Contains the class id to name mappings:  <class id>, <name>

train_test_split.txt Proposed split of images used for training and testing set:  <image id>, <is training?>
